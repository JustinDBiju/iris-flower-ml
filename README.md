# Iris Flower Species Prediction

A simple machine learning project that predicts the species of an Iris flower (Setosa, Versicolor, Virginica) based on sepal and petal measurements.

## Tools & Libraries
- Python
- scikit-learn
- pandas
- matplotlib / seaborn
